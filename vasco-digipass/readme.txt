A plugin to implement Vasco's MYDIGIPASS.com two-factor authentication service into your blog.

== Description ==

MyDIGIPASS.com enables web developers to add strong user authentication to their Wordpress site, in a cost-efficient and convenient way. Once added, users are able to secure access to their account with strong authentication to avoid account theft.
Additionally, you are able to tap into a community of millions of DIGIPASS users out there. Gain extra visibility by adding your site to theMyDIGIPASS.com application eco-system, making it easier to attract new users to read your blog.

For more information, please visit http://developer.mydigipass.com

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register your application on http://developer.mydigipass.com. The callback URL is http://yourdomain.com/index.php?mydigipass-action=callback
3. Fill in your *Client ID*, *Client Secret* and *Callback URL* 
Note that when you register the application on the developer site, you are using the sandbox environment.
When you are ready to go live  you can request your production environment details via the site.
